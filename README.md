## Data-driven network-wide traffic flow modelling: Are we ready for real-world applications?
Authors: Laura Gualda, Mohamed Abouelela, Ramandeep Singh, Constantinos Antoniou

> **Status:** Pre-publication. Please contact the author before citing or reusing any content.

### Overview
This repository contains the search methodology and screening steps for a systematic literature review on data-driven network-wide traffic flow modelling, covering publications from 1970 to 2025 (including).

### Repository Structure
```
├── scopus_query.ipynb   # Search steps and data visualisation
├── requirements.txt     # Python dependencies
└── outputs/             # Raw search exports
```

## How to reproduce the search

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Make sure you have a Scopus API key: https://dev.elsevier.com/
4. Open `scopus_query.ipynb` in Jupyter and run it with your own API key
